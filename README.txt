CODECRAFT – Cybersecurity Task 3

Project Title: Password Strength Checker

Project Description:
This project is a Python-based Password Strength Checker developed as part of the CodeCraft Cybersecurity tasks. The program evaluates the strength of a password based on multiple security criteria such as length, uppercase and lowercase letters, digits, and special characters.

Objective:
The goal of this project is to understand the importance of strong password policies and how complexity requirements enhance security against brute-force and dictionary attacks.

Features:
- Checks minimum length requirement (8 characters)
- Detects uppercase letters
- Detects lowercase letters
- Detects numbers
- Detects special characters
- Classifies password as Weak, Moderate, or Very Strong

How to Run:
1. Ensure Python is installed.
2. Open terminal in the project directory.
3. Run: python3 password_strength.py
4. Enter a password when prompted.

Examples:
1. Input : Horcrux!23
   Output: Password Strength: Strong (Score: 5/6)
Great! This is a strong password.

2. Input: Abc123
   Output: Password Strength: Medium (Score: 3/6)
Suggestions to improve:
- Too short! Use at least 8 characters (12+ is better).
- Add special characters like !@#$%.

3. Input: 12345
   Output: Password Strength: Weak (Score: 2/6)
Suggestions to improve:
- Too short! Use at least 8 characters (12+ is better).
- Add uppercase letters.
- Add special characters like !@#$%.

Learning Outcome:
Through this task, I gained practical knowledge of:
- Password security policies
- Regular expressions in Python
- Input validation techniques
- Basic security assessment logic
